二、子组件传父组件数据

JavaScript 设计模式： 触发事件 dispatchEvent | 监听事件 addEventListener

VueJs有类似模式：子组件触发事件 \$emit() | 父组件监听子组件 \$on()

【Demo实例 https://jsbin.com/homuwol/1/edit?html,output 】

三、组件中使用 v-model

v-model 是一个语法糖：

1､v-bind 绑定一个value 属性

2､ v-on 给 当前元素 绑定 input 事件

使用条件

1､接收一个value 属性

2､在有新的 value 时，触发 input 事件

【Demo实例 https://jsbin.com/yebahoy/edit?html,output 】

四、非父组件间通信

1､定义

两个组件间需要通信（非父子关系），在简单场景下，可用一个空的vue实例作为中央事件总线

2､操作流程 （点击A组件的东西，向B组件传递数据）

a、组件A中，\$emit 触发一个事件，并把参数传递过去

b、组件B中，在实例创建时监听这个事件，然后作为参数再传递进来就可以了

【Demo实例 https://jsbin.com/yebahoy/8/edit?html,output】

4､子链 (父实例访问子实例) this.$refs

子组件加索引：ref 属性 + 父组件拿子组件内容：this.$refs
