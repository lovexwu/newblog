#常用命令行如下
git status 查看本地git仓库的状态
git add -A 添加所有修改的内容
git commit -m “提交详情内容记录” 提交更新到本地仓库
git push -u origin master 将本地仓库更新的内容提交到远程仓库
git diff 查看修改的内容
git log 查看历史版本，显示从最近到最远的提交日志
